# Task 3: SQL for Data Analysis

## 📌 Objective

To perform basic data analysis using SQL concepts like SELECT, WHERE, GROUP BY, and ORDER BY.

## 🛠 Tools Used

* Microsoft Excel
* SQL (queries written in SQL file)

## 📂 Dataset Used

* olist_orders_dataset.xlsx

## 📊 Tasks Performed

1. **SELECT**

   * Viewed complete dataset

2. **WHERE**

   * Filtered orders where order_status = 'delivered'

3. **GROUP BY**

   * Counted number of orders for each order_status using Pivot Table

4. **ORDER BY**

   * Sorted dataset based on order_purchase_timestamp
